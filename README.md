### The 'Rick and Morty' GraphQL Playground is a development tool that provides a streamlined environment for exploring and interacting with the 'Rick and Morty' API using GraphQL.

### Features:
- authorization
- localization
- semantic layout
- private routes
- error boundary
- custom 404 page
- IDE integration

  ### Technologies:
- TypeScript
- React
- Redux Toolkit
- React Hook Form
- Firebase
- i18next
- Material UI
- Vite
- Vitest
- Eslint, Prettier, Husky
